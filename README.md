# Front-end Examine (React + Vite)

Test task: responsive layout + posts list from API, search, and post modal popup.

## Tech
- React (hooks)
- Vite
- CSS (no UI libraries)

## Setup
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Features
Sticky desktop navigation
Mobile menu 
CSS-only hover submenus 
Posts fetched from API
Search filtering by title/description
Post details popup 
Responsive posts grid 
Retina images via srcset
